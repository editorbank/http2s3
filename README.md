# The HTTP bridge to S3/AWS compatible storage

## Windows:
* Run
```
start .\bin\run
```
* Open
```
start http://localhost:81
```
* Clean
```
.\bin\clean
```


## Linux:

* Run
```
./bin/run.sh
```
* Open
http://localhost:81

* Clean
```
./bin/clean.sh
```

## Run via docker-compose
```
docker-compose up
```
